RacerPro Protégé Plugin

This plug-in makes the RacerPro reasoner available in Protégé

Plugin update

After updating the RacerPro Protégé Plugin and restarting Protégé a new binary version of RacerPro will be downloaded. Please make sure that there is an online connection available.
